Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Результат выполнения программы:
[3 2 3]

Объяснение:
Слайс представляет из себя структуру с тремя полями:
1. Указатель на подлежащий массив.
2. Len - длина слайса (сколько на данный момент слайс фактически содержит элементов)
3. Cap - емкость слайса (capacity), т.е. сколько фактически он способен вместить элементов до расширения.
При передаче слайса в качестве аргумента в функцию мы передаем указатель на данный слайс (копируем его)
в стек функции. Т.о. операции, выполняемые в теде функции, могут изменять исходный слайс,
переданный в качестве аргумента.

Т.о., при изменении слайса при помощи инструкции i[0] = "3", мы изменяем первый элемент подлежащего массива.
Слайс становится равным [3 2 3], len = 3, cap = 3.
Когда мы добавляем еще один элемент i = append(i, "4"), создается новый подлежащий массив,
равный [3 2 3 4], len = 4, cap = 6. Однако массив уже будет новым, данная операция никак не повлияет
на первоначальный массив, равный [3 2 3], также как и последующие операции. При этом выводится в консоль
будет именно первоначальный массив.
```
